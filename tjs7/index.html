SPEAKER #1
Anton Vorozheev
Software Engineer @ Malwarebytes
Migrating Legacy Systems With MicroFrontends.

My talk is dedicated to use of MicroFrontends (and specifically WebPack 5 Module Federation) to help engineers migrate legacy UI applications without sacrificing regular business deliveries. The following concerns are raised and addressed:

  • allowing several teams with different backgrounds contribute to the same UI
  • incremental merge of two separate legacy UI applications into one
  • gradual migration from the legacy codebase
  • some technical "magic" behind WebPack Module Federation and the ways of using it in development and production

SPEAKER #2
Bogdan Bashev
Staff Quality Engineer @ Veriff
Your ESLint - your (custom) rules

Custom ESlint rules allow you to save time in code review by automating checks for repeated problems and tailor code analysis to the needs of your team. In this talk, I will go over the whole process of creating a custom ESlint rule from idea to production.

I will cover:

  • Why would you want to use custom ESLint rules?
  • How to generate your own custom rule using Yeoman
  • How to parse syntax tree with AST Explorer and ESQuery
  • How to test and enable your new rule
